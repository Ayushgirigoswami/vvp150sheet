## 🌟 Why This Tracker?

Preparing for coding interviews? This isn't just another question list. It's your **complete interview preparation companion** with features that actually help you crack interviews:

- ✅ **Track every solved question** with timestamps
- 🔥 **Daily streak tracking** to build consistency
- 📊 **Weak area analysis** to focus your efforts
- 🎯 **Interview simulation mode** with timer

- 💾 **100% offline** - works without internet
- 🎨 **Beautiful dark UI** - easy on the eyes

## ✨ Features That Make a Difference

### 📈 Smart Progress Tracking
- **Checkbox System**: Mark questions as solved
- **Solved Date Display**: See "5 days ago", "2 weeks ago" for each question
- **Progress Percentage**: Real-time completion stats
- **Topic-wise Progress**: Track progress per topic with visual bars

### 🔥 Daily Streak System
- Tracks consecutive days of solving
- Motivates daily practice (crucial for interviews!)
- Shows last solved date
- Builds discipline and consistency

### 🎯 Interview Preparation Tools

#### 🎲 Random Question Mode
- Pick random unsolved question
- Shows question details in clean popup
- Quick "Mark Solved" and "Add Note" actions
- Perfect for daily practice

#### ⏱️ Interview Simulation Mode
- Simulates real interview environment
- Timer tracks solving time
- 5 HIGH priority questions per session
- Records time spent per question
- Builds time management skills

### 📊 Weak Areas Analysis
- Automatically identifies topics with <50% completion
- Visual progress bars per topic
- Helps prioritize study time
- Focus on what matters most

### 📝 Notes & Code Storage
- Add notes/solutions for each question
- Rich text support
- Searchable notes
- **Export all notes as PDF** for revision

### 🔖 Revision System
- Mark important questions for revision
- One-click access to revision list
- Perfect for last-minute prep
- Review difficult questions easily

### 🔍 Advanced Filtering
- **Search by name**: Find questions instantly
- **Filter by topic**: BASICS, ARRAY, STRING, etc.
- **Filter by difficulty**: Easy, Medium, Hard
- **Filter by pattern**: Two Pointer, Sliding Window, etc.
- **Filter by company**: TCS, Infosys, Wipro, L&T
- **Revision filter**: Show only marked questions

### 📄 PDF Export
- Export all notes as professional PDF
- Organized by topics
- Includes question details, patterns, companies
- Perfect for printing and offline study
- Auto-dated filename

## 🎓 How to Use Effectively

### For Beginners
1. Start with **BASICS** topic
2. Solve in order (Easy → Medium → Hard)
3. Add notes for every solution
4. Mark difficult questions for revision
5. Aim for 3-5 questions daily

### For Interview Prep (1-2 months)
1. Filter by **company** you're targeting
2. Focus on **HIGH priority** questions
3. Use **Interview Mode** for timed practice
4. Review **Weak Areas** weekly
5. Maintain **daily streak** for consistency

### Last Week Before Interview
1. Click **Revision List** button
2. Review all marked questions
3. Export notes as PDF for quick reading
4. Do 2-3 **Interview Mode** sessions daily
5. Focus on **HIGH priority** unsolved questions
